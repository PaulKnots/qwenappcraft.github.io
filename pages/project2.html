Weather Forecast

A beautiful and functional weather application that provides real-time forecasts for any location worldwide. Built with HTML, CSS, and JavaScript using a public weather API.

Technologies: HTML5, CSS3, JavaScript, Weather API

Features: Geolocation, 5-day forecast, Weather animations, Responsive design

Project Overview

The Weather Forecast app was created to provide users with accurate and visually appealing weather information. The application automatically detects the user's location or allows them to search for any city worldwide.

Using a public weather API, the app displays current conditions along with a 5-day forecast. The interface features dynamic background animations that change based on weather conditions (sunny, rainy, cloudy, etc.).

Special attention was paid to the user experience, with intuitive controls and clear visual hierarchy. The responsive design ensures the app works perfectly on all device sizes, from smartphones to desktop monitors.
