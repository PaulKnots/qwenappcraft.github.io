Task Manager Pro

A comprehensive task management solution built entirely with HTML, CSS, and JavaScript. This application helps users organize their daily activities, set priorities, and track progress.

Technologies: HTML5, CSS3, JavaScript

Features: Drag & drop, Local storage, Priority levels, Due dates

Project Overview

Task Manager Pro was designed to solve the common problem of disorganization in daily life. The application provides a clean interface where users can create, edit, and delete tasks with ease. Tasks can be categorized by priority level (high, medium, low) and assigned due dates.

The drag-and-drop functionality allows users to rearrange tasks according to their workflow. All data is stored in the browser's local storage, ensuring that tasks persist between sessions without requiring a server backend.

The responsive design ensures the application works seamlessly on both desktop and mobile devices, making it accessible anytime, anywhere.
